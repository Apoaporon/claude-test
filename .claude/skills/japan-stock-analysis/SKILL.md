# 日本株 個別銘柄 調査・分析スキル

## 概要

ユーザーが指定した日本株（個別銘柄）について、ウェブ検索で最新情報を収集し、長期投資（バリュー・成長株）の視点から構造化されたレポートを作成する。

---

## ワークフロー

### Step 1: 銘柄の特定

ユーザーの入力から以下を確認する：
- 銘柄名 or 証券コード（例：トヨタ自動車、7203）
- 不明な場合は確認する

### Step 2: 情報収集（Web検索）

以下の情報を並列で収集する（web_search ツールを使用）：

1. **企業概要・事業内容**
   - 検索例：`{銘柄名} 事業内容 ビジネスモデル`
   - 主力事業、セグメント、競合優位性

2. **最新業績・財務情報**
   - 検索例：`{銘柄名} 決算 売上 営業利益 2024 2025`
   - 直近の売上高、営業利益、純利益のトレンド

3. **バリュエーション指標**
   - 検索例：`{銘柄名} PER PBR ROE 配当利回り`
   - PER、PBR、ROE、配当利回り、時価総額

4. **配当情報**
   - 検索例：`{銘柄名} 配当 配当利回り 配当金 増配 減配 歴史`
   - 配当利回り、1株配当額、配当性向、増配・減配の履歴、連続増配年数

5. **株価・チャート動向**
   - 検索例：`{銘柄名} 株価 チャート 52週高値安値 2025`
   - 直近株価、52週高値・安値、移動平均との関係、直近の株価トレンド（上昇・下落・横ばい）

6. **最新ニュース・IR情報**
   - 検索例：`{銘柄名} ニュース IR 2025`
   - 直近3〜6ヶ月の重要ニュース、決算発表、中期経営計画

7. **業界・競合ポジション**
   - 検索例：`{銘柄名} 競合 市場シェア 業界`

収集した情報を以下のフォーマットで出力する。

### Step 4: ファイル保存とPDF出力

レポートを以下の手順で保存・変換する：

1. **Markdownファイルとして保存**
   - 保存先：`reports/{銘柄名}_{YYYYMMDD}.md`（プロジェクトルート直下の `reports/` ディレクトリ）
   - 例：`reports/NTT_20260331.md`
   - `reports/` ディレクトリが存在しない場合は作成する

2. **PDF変換スクリプトを呼び出す**
   ```bash
   python ${CLAUDE_SKILL_DIR}/scripts/report_to_pdf.py reports/{銘柄名}_{YYYYMMDD}.md
   ```
   - 出力先：`reports/{銘柄名}_{YYYYMMDD}.pdf`（自動的に同名の.pdfが生成される）

3. **依存ライブラリの確認**
   - スクリプト実行前に `markdown` と `weasyprint` がインストール済みか確認する
   - 未インストールの場合はユーザーに通知し、以下を案内する：
     ```bash
     pip install markdown xhtml2pdf
     ```

4. **完了通知**
   - 保存されたMarkdownファイルのパスとPDFファイルのパスをユーザーに伝える

## 注意事項

- **数値が取得できない場合**は「情報なし」と明記し、推測で埋めない
- **最新性を重視**：検索結果の日付を確認し、古い情報には注記する
- **免責事項を必ず記載**：投資助言にならないよう注意
- バリュエーション指標は業界・セクターによって適正水準が異なるため、業界文脈を添える
- ユーザーが複数銘柄を比較したい場合は、銘柄ごとにレポートを作成するか、比較表を作成する
