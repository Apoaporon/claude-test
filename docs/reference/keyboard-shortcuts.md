# キーボードショートカット

Claude Codeのインタラクティブモードで使用できるキーボードショートカット一覧です。

## 基本操作

| ショートカット | 動作 |
|---|---|
| `Enter` | メッセージを送信 |
| `Shift + Enter` | 改行（複数行入力） |
| `Ctrl + C` | 現在の操作を中断 |
| `Ctrl + D` | Claude Codeを終了 |
| `Ctrl + L` | 画面をクリア |

## 履歴操作

| ショートカット | 動作 |
|---|---|
| `↑` (上矢印) | 前のメッセージに戻る |
| `↓` (下矢印) | 次のメッセージに進む |
| `Ctrl + R` | 履歴を検索 |

## テキスト編集

| ショートカット | 動作 |
|---|---|
| `Ctrl + A` | 行頭に移動 |
| `Ctrl + E` | 行末に移動 |
| `Ctrl + U` | カーソルより前を削除 |
| `Ctrl + K` | カーソルより後を削除 |
| `Ctrl + W` | 前の単語を削除 |
| `Alt + B` | 前の単語に移動 |
| `Alt + F` | 次の単語に移動 |

## ツール承認

Claude Codeがツール使用の確認を求めた際:

| ショートカット | 動作 |
|---|---|
| `y` / `Enter` | 承認 |
| `n` | 拒否 |
| `a` | 常に承認（セッション中） |
| `d` | 詳細を表示 |

## エディタ統合

VS CodeのTerminalでClaude Codeを使用する場合、VS Codeのショートカットも有効です。

| ショートカット | 動作 |
|---|---|
| `Ctrl + `` ` | ターミナルのフォーカス切り替え |
| `Ctrl + Shift + 5` | ターミナルを分割 |

## カスタマイズ

`~/.claude/keybindings.json` でキーバインドをカスタマイズできます。

```json
{
  "submit": "ctrl+enter",
  "newline": "enter"
}
```

`Enter` で改行、`Ctrl + Enter` で送信する設定に変更できます。
