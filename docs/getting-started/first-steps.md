# 最初のステップ

## プロジェクトで使ってみる

Claude Codeは、プロジェクトディレクトリから起動することで、コードベースを認識します。

```bash
cd my-project
claude
```

## インタラクティブモード

引数なしで起動すると、インタラクティブモードになります。

```
> バグを修正してください
> src/utils.ts の型エラーを直してください
> テストを追加してください
```

## 一回限りの実行

特定のタスクをコマンドラインから直接実行できます。

```bash
claude "src/App.tsx のコンポーネントを説明してください"
claude "README.md を日本語で作成してください"
```

## ファイルを指定する

`@` 記号でファイルを参照できます。

```
> @src/main.py のロジックを最適化してください
> @tests/test_api.py のテストケースを追加してください
```

## 基本的なワークフロー

1. プロジェクトディレクトリに移動
2. `claude` を起動
3. 自然言語でタスクを指示
4. 変更内容を確認・承認
5. 必要に応じて修正を依頼

## ヒント

- 具体的な指示を出すほど、より正確な結果が得られます
- 変更を加える前に、Claudeは変更内容を説明します
- `Ctrl+C` でいつでも中断できます
